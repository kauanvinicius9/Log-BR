# Logistics Management

Handling and processing **logistics data** using a database for data creation and analysis, including cleaning null and duplicate values, and correcting any errors using **KNIME**, a **Low Code** tool for automating processes developed across various tools. This analysis also includes organized reports with interactive **dashboards**. Check it out!

To access **Power BI**, just download a copy of the `GestaoLog.pbix` file.
